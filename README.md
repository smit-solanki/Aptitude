A professional Android application built for students preparing for competitive exams like SSC, Banking, Railways, and IndiaBIX. This app features a categorized quiz system, real-time result analysis, and a randomized "Mixed Quiz" mode.

## üöÄ Features
* **Topic-Wise Learning:** Dedicated sections for Chain Rule, Time and Work, Boats and Streams, Trains, and more.
* **Mixed Quiz Mode:** A mega-exam mode that randomly selects 15 questions from all topics to simulate real exam pressure.
* **Dynamic Progress Tracking:** Visual Progress Bar to track your status during the quiz.
* **Result Analysis:** A detailed review screen showing your score, the answers you picked (in Red/Green), and the correct solutions.
* **Modern UI:** Built using Google Material Design components (MaterialCardView, CircularProgressIndicator).
* **Monetization Ready:** Fully integrated with Google AdMob Banner Ads.
